DashX360 is a fan-made Windows recreation of the Xbox 360 Metro dashboard. The goal is to preserve the feel of the original interface: big tiles, controller navigation, Guide overlays, and the sharp panel-based layout that defined that era of the dashboard.


## Version 1.1 New Features

    ## Features

    - Xbox 360-inspired dashboard tabs for games, apps, music, video, social, and settings
    - Controller-first navigation with keyboard and mouse support
    - Xbox Guide overlay with Friends, Party, Profile, and search screens
    - Local profile and friend data with cached gamer pictures
    - Boot video, dashboard audio cues, and Metro-style tile presentation
    - Custom theme support, and the Bing Penguins are Back!
    - Library Scan for Steam, and uses coverart steam provides
    - Import and Export for user data for transfer or version updating


    ## Controls

    - `A / Enter` selects
    - `B / Escape` goes back
    - `X` triggers context actions where available
    - `Y` triggers secondary actions where available
    - Guide combo / hotkey opens the Xbox Guide overlay
    - Mouse support is available for tiles, buttons, and popup menus

    ## Building

    ### Requirements

    - Untick `Enable Guide Button Chords for controllers` to use the guide if using steam
    - Windows 10 or Windows 11
    - .NET 8 SDK
    - Visual Studio 2022 or the .NET CLI

    ## Configuration Notes

    - Launcher settings and cached profile data are all now stored stored in the DashX360s folder
    - This public release uses local profile and friend data only
    - Friends, profiles, and party screens are included, but they run on local data in this version
